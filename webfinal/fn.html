JavaScript String and Math Functions

String Functions

  • length: Returns the length of a string.
    • let str = "Hello, World!";
    let len = str.length;

    Result: Length of string is 13

  • toUpperCase(): Converts all characters in a string to uppercase.
    • let upper = str.toUpperCase();

    Result: HELLO, WORLD!

  • toLowerCase(): Converts all characters in a string to lowercase.
    • let lower = str.toLowerCase();

    Result: hello, world!

  • substring(start, end): Extracts characters from a string between two indices.
    • let sub = str.substring(0, 5);

    Result: Hello

  • includes(): Checks if a string contains a specified substring.
    • let hasWorld = str.includes("World");

    Result: true

Math Functions

  • Math.abs(): Returns the absolute value of a number.
    • let abs = Math.abs(-42);

    Result: 42

  • Math.round(): Rounds a number to the nearest integer.
    • let rounded = Math.round(4.7);

    Result: 5

  • Math.sqrt(): Returns the square root of a number.
    • let squareRoot = Math.sqrt(64);

    Result: 8

  • Math.pow(base, exponent): Returns the base to the power of the exponent.
    • let power = Math.pow(2, 3);

    Result: 8

  • Math.random(): Generates a random number between 0 and 1.
    • let randomNum = Math.random();

    Result: 0.534 (example)
